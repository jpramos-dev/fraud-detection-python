# Análise de Padrões e Detecção de Fraudes utilizando Python

Projeto de análise exploratória e detecção de fraudes em transações bancárias utilizando Python, técnicas estatísticas e Machine Learning.

## Objetivos
- Explorar dados transacionais
- Identificar padrões de comportamento
- Detectar anomalias com Z-Score e IQR
- Aplicar Isolation Forest para detecção de fraudes

## Tecnologias
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Dataset
O dataset utilizado está disponível neste repositório para fins educacionais.
